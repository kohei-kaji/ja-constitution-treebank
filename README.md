# UD_Japanese-Constitution

> [!WARNING]
> **WORK IN PROGRESS**

This repository provides the Universal Dependencies (UD) annotation for [the Constitution of Japan](https://www.shugiin.go.jp/internet/itdb_annai.nsf/html/statics/shiryo/dl-constitution.htm).

Note: currently, the annotation work covers only the preamble.

The text was initially parsed using [GiNZA](https://megagonlabs.github.io/ginza/) via [spaCy](https://spacy.io/) and subsequently refined through manual correction.

## 📁 Repository Contents

*   **`preamble.txt`**: The original Japanese text of the preamble.
*   **`preamble-en.txt`**: The English translation of the preamble.
*   **`constitution.txt`**: The original Japanese text of the constitution.
*   **`silver/preamble-ginza.conllu` & `silver/constitution-ginza.conllu`**: The initial, automatically parsed data generated by GiNZA (silver standard).
*   **`preamble-gold.conllu`**: The manually corrected, gold-standard UD annotation.
*   **`depparse.py` & `parse.sh`**: The Python and shell scripts used to run the initial dependency parsing.
*   **`generate_figures.py`**: A script to generate dependency tree visualizations from the gold annotation.
*   **`figure/`**: A directory containing the generated `.tex` and `.pdf` visualization files, rendered using the LaTeX `tikz-dependency` package.

## ⚙️ Project Workflow

1.  **Automated Parsing**: The original text (`preamble.txt`) was parsed by running `parse.sh`, which executes `depparse.py` utilizing the GiNZA NLP library. The resulting output is stored as `silver/preamble-ginza.conllu`.
2.  **Manual Annotation**: The silver data was manually corrected to `preamble-gold.conllu`.
3.  **Visualization**: The `generate_figures.py` script was executed against the gold data to generate syntax dependency trees. These are output as LaTeX files and compiled into PDFs inside the `figure/` directory.
